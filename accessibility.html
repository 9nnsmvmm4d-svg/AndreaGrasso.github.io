Accessibility Statement

This website aims to be usable by as many people as possible, including people using assistive technologies or alternative ways of navigating the web.

Accessibility approach

This website is designed with attention to readable text, clear structure, keyboard-friendly navigation, and responsive layouts across device sizes.

What has been considered

  • Clear page structure and headings.
  • Responsive layouts for mobile and desktop.
  • Accessible navigation, including button-based mobile menu behaviour.
  • Colour contrast and readable text sizes where possible.
  • Alternative text for meaningful images where used.

Known limitations

Some parts of the website may not yet be fully optimised for every assistive technology, browser, or accessibility need. Accessibility work is ongoing and improvements may continue over time. GOV.UK and DWP accessibility guidance recommend tailoring an accessibility statement to the actual state of the site rather than copying a generic statement. [web:173][web:176]

Feedback and contact

If you experience difficulty using this website, notice an accessibility issue, or need information in a different format, please contact: hello@andreagrasso.co.uk.

Review and updates

This accessibility statement may be updated as the website changes and accessibility improvements are made.
